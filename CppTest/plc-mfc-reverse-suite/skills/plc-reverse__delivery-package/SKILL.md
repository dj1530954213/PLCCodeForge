# plc-reverse:delivery-package

## 目标
将零散的脚本、Hook DLL、笔记整理成最终产品。

## 交付清单
1. **Injector & Payload**：一键启动器，自动启动目标软件并注入功能增强 DLL。
2. **SDK / API 文档**：如果你导出了内部函数，提供头文件和库文件。
3. **Tools**：独立的工程文件解析器、协议测试工具。
4. **Patch**：如果进行了二进制修改，提供 Patcher 工具。
